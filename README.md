# 🌈 小马宝莉魔法计时器 | My Little Pony Style Timer

> 一款甜美梦幻、带有彩虹小马风格的交互式计时器。  
> 按下空格键开始魔法时刻，再次按下暂停，记录每一个友谊与成长的瞬间。

## ✨ 特色功能

- 🎮 **键盘核心控制**  
  - <kbd>空格键</kbd> → 开始 / 暂停计时  
  - <kbd>R 键</kbd> → 记录当前时刻（圈点/时间节点）  
  - <kbd>ESC 键</kbd> → 一键重置计时器并清空记录  

- 🦄 **小马宝莉魔法视觉**  
  - 彩虹渐变数字 + 糖果色按钮  
  - 漂浮爱心、独角兽角装饰、云朵日记本风格记录面板  
  - 点击区域可能触发随机彩蛋特效 🌟  

- 📖 **“友谊时光册”记录系统**  
  - 每次按「记录圈点」或 <kbd>R 键</kbd> ，当前时间会保存在列表中  
  - 可随时「清空记录」，重新开始记录魔法时刻  
  - 记录条目带有小马图标，滚动流畅，最多可保存数十条  

- 🎨 **完美响应式**  
  - 手机、平板、PC 均自适应布局  
  - 保留了毫秒级精确计时，基于 `requestAnimationFrame` 和 `Date.now()` 差值，即使页面后台也能保证时间准确
